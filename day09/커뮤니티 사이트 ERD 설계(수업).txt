<커뮤니티 사이트 ERD-수업>
1. 요구사항
- 관리자는 게시글 종류를 등록한다. 
- 게시글은 로그인한 회원만 작성이 가능하다.
- 게시글은 제목, 내용을 입력하고, 첨부파일은 최대 3개 선택 가능하다.
- 문의게시글에는 관리자가 문의에 대한 답변 게시글을 작성한다.
- 게시글에는 로그인한 회원만 댓글, 대댓글, 대대댓글 등을 달 수 있다.
- 게시글은 작성일 기준으로 최신순으로 정렬한다.
- 댓글은 작성일 기준으로 최신순으로 정렬한다.
- 비밀글은 관리자와 작성자만 본다
- 추천/비추천은 로그인한 회원만 가능하다
- 한 게시글에 대해 추천/비추천은 하나만 가능하다
- 한 게시글에 대해서 추천/비추천은 한번만 가능하다
- 게시글 조회는 비회원도 가능하다
---------------------------------------------------------------------------------------
2. ERD 설계
- 회원
= 회원가입시 아이디, 비번, 비번확인, 이름, 전화번호, 주소 입력
 -> 아이디 비번 비번확인 이름 전화번호는 필수항목이다
 -> 전화번호는 -를 포함한 11자리 숫자를 입력한다.(예 : 010-4227-0852)
 -> 주소는 선택항목이다
  => 주소는 주소와 상세를 함께 관리한다
= 회원 아이디는 8자에서 20자이하이며, 영문자,숫자, !@#$%^&*만 가능하다
= 회원 비밀번호는 8자에서 15자자 이하이며 영문자, 숫자 !@#$%^&*만 가능하다
= 회원 권한은 0~10이고, 10은 슈퍼관리자, 1은 일반회원으로 한다. 0은 비회원
---------------------------------------------------------------------------------------
- 문자열 비교보다 정수 비교가 빠르다
- null보다는 기본값을 설정하는게 좋다
---------------------------------------------------------------------------------------
- 게시글
= 게시글 삭제 시 작성자가 삭제하면 1, 관리자가 삭제하면 2, 0이면 삭제가 안된 상태
= 부모번호는 답글들의 최고 부모번호, 본문게시글의 각 기본키
= 답변게시글, 대댓
1. 	제목1			내용1		abc		1	1
2.	답변:제목1		내용2		관리자	1	2
3. 	제목2			내용3		qwe		2	1
4.	답변:제목1		내용4		관리자	1	2
5. 	답변:답변:제목1	내용5		abc		1	3
= 비밀여부 0이면 공개, 1이면 비밀글
---------------------------------------------------------------------------------------
- 추천/비추천
= 게시글과 댓글에 추천/비추천한다(상태 - 추천: 1, 비추천: -1, 아무것도 선택 안함: 0)
= 추천을 취소하면 상태가 0으로 된다. 
---------------------------------------------------------------------------------------
- 댓글
= 비밀여부 0이면 공개, 1이면 비밀글
= 댓글 삭제 시 작성자가 삭제하면 1, 관리자가 삭제하면 2, 0이면 삭제가 안된 상태
---------------------------------------------------------------------------------------
- 카테고리
= 권한은 회원등급을 적어 일정 등급 이상만 읽거나 작성할 수 있게 한다.
